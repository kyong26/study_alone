# 2. LIS

LIS : Longest Increasing Subsequence(최대 부분 증가 수열)

N개의 자연수로 이루어진 수열이 주어졌을 때, 그 중에서 가장 길게 증가하는(작은 수에서 큰 수로) 원소들의 집합을 찾는 프로그램을 작성하라.
예를 들어, 원소가 2, 7, 5, 8, 6, 4, 7, 12, 3이면 가장 길게 증가하도록 원소들을 차례대로 뽑아내면 2, 5, 6, 7, 12를 뽑아내어 길이가 5인 최대 부분 증가수열을 만들 수 있다.

- 문자 배열의 순서를 바꾸지 말아야 함

- **입력설명**
첫째 줄은 입력되는 데이터의 수(N은 1 이상, 1000 이하)를 의미하고,
.둘째 줄은 N개의 입력데이터들이 주어진다.
- **출력설명**
첫 번째 줄에 부분증가수열의 최대 길이를 출력한다.

- **입력예제**
8
5 3 7 8 6 2 9 4
- **출력예제**
4



```python
n = 8
arr = [5,3,7,8,6,2,9,4]

arr.insert(0,0)
dy =[0]*(n+1)
res=0
for i in range(2, n+1): #4,5,6,7,8
    max=0
    for j in range(i-1, 0, -1):
        if arr[j]<arr[i] and dy[j]>max: #항의 가장 마지막 숫자를 찾는다
            max=dy[j]
    dy[i] = max+1
    if dy[i]>res:
        res=dy[i]
print(res)
```

    4
    

## 응용문제: 가장 높은 탑 쌓기

밑면이 정사각형인 직육면체 벽돌들을 사용하여 탑을 쌓고자 한다. 탑은 벽돌을 한 개씩 아래에서 위로 쌓으면서 만들어간다. 아래 조건을 만족하면서 가장 높은 탑을 쌓을 수 있는 프로그램을 작성하시오.

(조건1) 벽돌은 회전시킬 수 없다. 즉 옆면을 밑면으로 사용할 수 없다.
(조건2) 밑면의 넓이가 같은 벽돌은 없으며, 또한 무게가 같은 벽돌도 없다.
(조건3) 벽돌들의 높이는 같을 수도 있다.
(조건4) 탑을 쌓을 때 밑면이 좁은 벽돌 위에 밑면이 넓은 벽돌은 놓을 수 없다.
(조건5) 무게가 무거운 벽돌을 무게가 가벼운 벽돌 위에 놓을 수 없다.

- 입력설명

입력 파일의 첫째 줄에는 입력될 벽돌의 수가 주어진다. 입력으로 주어지는 벽돌의 수는 최대 100개이다. 둘째 줄부터는 각 줄에 한 개의 벽돌에 관한 정보인 벽돌의 밑면의 넓이, 벽돌의 높이, 그리고 무게가 차례대로 양의 정수로 주어진다. 각 벽돌은 입력되는 순서대로 1부터 연속적인 번호를 가진다.

- 입력예시

```
n = 5
arr = [[25, 3, 4], [4, 4, 6], [9, 2, 3], [16, 2, 5], [1, 5, 2]]
```

- 출력예시

```
10
```



```python
n = 5
arr = [[25, 3, 4], [4, 4, 6], [9, 2, 3], [16, 2, 5], [1, 5, 2]]

dy = [0] * (n+1)
arr.sort(reverse=True)
arr.insert(0, [0,0,0])

dy[1] = arr[1][1]

res = 0

for i in range(2, n+1):
    max_value = 0
    for j in range(i-1, 0, -1):
        if arr[j][0]>arr[i][0] and arr[j][2]>arr[i][2] and dy[j]>max_value:
            max_value = dy[j]
        dy[i] = max_value + arr[i][1]
        if dy[i]>res:
            res = dy[i]
print(res)
```

    10
